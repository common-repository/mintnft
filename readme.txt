== Description ==

 The “MintNFT Plugin” is a useful tool for those who want to mint their own NFTs without having to build an entire minting platform from scratch. With this plugin, you can easily add all the necessary metadata and NFT files from the backend, as well as add all the relevant information for your NFTs. Once you have completed these steps, you can easily deploy your contract using the plugin. This simplifies the process of creating and deploying NFTs, making it more accessible to a wider range of users. 
 
#### Detailed Description: 

The MintNFT Plugin is a user-friendly tool designed for WordPress site owners who want to mint NFTs without any complicated technical skills. With this plugin, you can easily publish your art on IPFS/Pinata and create NFT tokens when a buyer purchases your NFTs by paying the set NFT price and gas fees. 

Adding the Minting functionality page to your website is a breeze and can be done in just a few seconds. All you need to do is install the plugin on your WordPress site, follow the step by step easy to follow process, and credit your crypto wallet with enough Native currency (i.e.: ETH, MATIC etc..) balance for the selected Blockchain. Once you have done that, you are ready to start minting your own NFTs. Overall, the NFT Minting plugin is an efficient and convenient way to create NFT minting platform, even if you have limited technical knowledge.

#### Feature List:
* The Owner can upload the number of NFTs that are ready to mint and should be displayed to the end customer. With the Metamask wallet up and ready, one can easily mint the NFTs by just clicking on the “Mint” Button, adding the amount of NFTs to be minted, and voila, the transaction will be completed within no time. 
* You can easily upload the digital copies of JSON metadata file implementation according to metadata standards, so, they can be accessed for minting. 
* NFT token content is protected by cryptographically generated hashes of all files and attributes attached. The token URI itself is a hash of the JSON metadata file. 
* Automatic storage of metadata to IPFS/Pinata 
* Tokens can be minted from the vendor or admin address. NFT token smart contract can be deployed from the WordPress admin dashboard. 
* When someone buys your NFT, the NFT is minted to the wallet address connected with metamask at checkout.

== Installation ==

### Automatic installation
1. Search for mintnft through 'Plugins > Add New' interface.
2. Find the plugin box of mintnft and click on the 'Install Now' button.
3. Then activate the mintnft plugin.

### Manual installation

1. Download [mintnft](https://downloads.wordpress.org/plugin/mintnft.zip)
2. Upload the mintnft through 'Plugins > Add New > Upload' interface or upload mintnft folder to the '/wp-content/plugins/' directory.
3. Activate the mintnft plugin through the 'Plugins' menu in WordPress.
